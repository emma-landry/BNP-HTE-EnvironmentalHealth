## Reproducing the results in the Direct Accountability: Evaluating Specific Interactions section of the manuscript
- `scenario1_function.R`: function to simulate data for Scenario 1, as defined in the manuscript.

- `scenario1_simulate.R`: code to simulate data for Scenario 1, run simulations using the BPCF and CASBAH models.

- `scenario2_simulate.R`: code to simulate data for Scenario 2, run simulations using the BPCF and CASBAH models.

- `results.R`: code to reproduce the figures presented in the manuscript.

- The `.RData` files are the simulated datasets, as well as simulation results for both scenarios, with both methods.

## Reproducing the results in the Indirect Accountability: Exposure-Response Heterogeneity section of the manuscript
We note that the scenarios are flipped in the simulation files, but the figures are obtained such that Scenario 1 corresponds to the data generation process given by "Bayesian Regression Tree Models for Causal
Inference: Regularization, Confounding, and Heterogeneous Effect" and Scenario 2 corresponds to the data generation process given by "Confounder-dependent Bayesian mixture model: Characterizing heterogeneity of causal
effects in air pollution epidemiology".

- `scenario2_function.R`: function to simulate data for Scenario 1, as defined in the manuscript.

- `scenario1_simulate.R`: code to simulate data for Scenario 2, run simulations using the BPCF and CASBAH models.

- `scenario2_simulate.R`: code to simulate data for Scenario 1, run simulations using the BPCF and CASBAH models.

- `results.R`: code to reproduce the figures presented in the manuscript.

- The `.RData` files are the simulated datasets, as well as simulation results for both scenarios, with both methods.
