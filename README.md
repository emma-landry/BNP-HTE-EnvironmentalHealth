# BNP-HTE-EnvironmentalHeath
In this repository, we provide code to reproduce simulation results in the paper "Bayesian Causal Methods for Heterogeneous Treatment Effects in Environmental Health" by D. Zorzetto, E. Landry, M. Guindani, D. Telesca, V. C. Edefonti, R. De Vito, and F. J. Bargagli-Stoffi.

## Code files
### Methods code in the **'code'** folder:
- **'BPCF'** folder: contains the necessary code from the <a href=https://github.com/lit777/BPCF/tree/main> _BPCF repository_</a>, associated with the paper <a href=https://academic.oup.com/biometrics/article/81/1/ujaf024/8081643> _"Bayesian nonparametric trees for principal causal effects"_</a>, by C. Kim and C. Zigler.

- **'HTEBayes'** folder: contains the necessary code from the <a href=https://github.com/dafzorzetto/HTEBayes> _HTEBayes repository_</a>, associated with the paper <a href=https://academic.oup.com/biometrics/article/80/2/ujae025/7652283#447330610> _"Confounder-dependent Bayesian mixture model: Characterizing heterogeneity of causal effects in air pollution epidemiology"_</a>, by D. Zorzetto, F. J. Bargagli-Stoffi, A. Canale, and F. Dominici.
 
- **'PSBayes'** folder: contains the necessary code from the <a href=https://github.com/dafzorzetto/PSBayes/tree/main> _PSBayes repository_</a>, associated with the paper <a href=https://arxiv.org/pdf/2405.17669> _"Bayesian Nonparametrics for Principal Stratification with Continuous Post-Treatment Variable"_</a>, by D. Zorzetto, A. Canale, F. Mealli, F. Dominici, and F. J. Bargagli-Stoffi.

### Simulation replications code i the **'simulations'** folder:
- **'Direct Accountability'** folder: contains code to reproduce results from the "Direct Accountability: Evaluating Specific Interactions" section of the manuscript.

- **'Indirect Accountability'** folder: contains code to reproduce results from the "Indirect Accountability: Exposrue-Response Heterogeneity" section of the manuscript.
